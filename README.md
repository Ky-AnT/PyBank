# PyBank
### Description:
PyBank is a Terminal application written in python, for managing and tracking the money that you have
### Instructions
Up & Down arrow keys to choose option
Shift to select option
### Installing Modules:
    python3 -m pip install -r requirements.txt
### Running the Program
    sudo python3
